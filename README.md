# Theory and Practice of Security – Hands-on Labs

This repository contains practical exercises completed as part of the
Theory and Practice of Security course.

The objective of these labs is to explore practical cybersecurity concepts
including secure configuration, credential protection, cryptographic primitives,
and vulnerability detection using static and dynamic analysis tools.

## Topics Covered

- Secure software configuration practices
- Credential protection in Spring Boot applications
- Cryptographic prime number generation
- Static application security testing (Semgrep)
- Dynamic application security testing (OWASP ZAP)
- Vulnerability identification and mitigation

## Tools Used

- Java / Spring Boot
- Python
- Semgrep
- OWASP ZAP
- GitHub
